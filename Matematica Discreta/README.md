# 🔢 Matemática Discreta

Bienvenido a mi repositorio, un espacio donde documento mi recorrido por la asignatura de **Matemática Discreta** del **Grado en Ingeniería Matemática (iMAT)** en **ICAI**.
Aquí reúno **todas las prácticas, ejercicios, demostraciones, algoritmos y entregables** que he desarrollado a lo largo del curso.

Este repositorio refleja tanto mi **progreso técnico** en el razonamiento matemático discreto como mi **crecimiento personal y metodológico** al aprender a estructurar el pensamiento lógico y abstracto.

---

## 🎯 Sobre la Asignatura

La asignatura de **Matemática Discreta** tiene como objetivo principal enseñar los fundamentos matemáticos que sustentan la **ciencia de la computación y la ingeniería**.
En el marco del grado iMAT, combina el rigor matemático con aplicaciones prácticas en informática, fomentando una comprensión profunda de las estructuras discretas y su relevancia en el mundo digital.

Durante el curso se trabajan aspectos como:

* 🧩 **Teoría de conjuntos** y relaciones binarias
* 🔗 **Lógica matemática** y métodos de demostración
* 📊 **Combinatoria** y técnicas de conteo
* 🪮 **Teoría de grafos** y aplicaciones
* 🔢 **Teoría de números** y criptografía básica

Cada práctica busca desarrollar una habilidad concreta —desde construir demostraciones formales hasta implementar algoritmos sobre estructuras discretas.

## 🧠 Aprendizajes Clave

Durante el curso he aprendido que **la matemática discreta no es solo teoría abstracta**, sino **la base fundamental de la computación moderna**: entender cómo se estructuran los datos, cómo se diseñan los algoritmos y cómo se demuestra su correctitud.

Algunos aprendizajes esenciales que me llevo son:

* La importancia del **rigor lógico** en la demostración de propiedades
* La **versatilidad de los grafos** para modelar problemas reales
* El valor de las **técnicas combinatorias** en el análisis de algoritmos
* La **relevancia de la teoría de números** en seguridad informática
* Y, sobre todo, que cada problema discreto enseña a pensar de manera estructurada y sistemática

## ⚙️ Requisitos y Uso

Para ejecutar los algoritmos y visualizar las implementaciones:

1. Clona el repositorio:

   ```bash
   git clone https://github.com/tuusuario/matematica-discreta-imat.git
   cd matematica-discreta-imat
   ```

2. Instala las dependencias básicas:

   ```bash
   pip install numpy matplotlib networkx
   ```

3. Ejecuta los algoritmos de grafos:

   ```bash
   python Practicas/Principales/P1 - Grafos/bfs_dfs.py
   ```

4. Para las demostraciones y documentos teóricos:

   ```bash
   open Practicas/Guiadas/Logica-Demostraciones/demostracion_directa.md
   ```
