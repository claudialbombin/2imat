# 📊 Visualización de Datos

Bienvenido a mi repositorio personal, un espacio donde documento mi recorrido por la asignatura de **Visualización de Datos** del **Grado en Ingeniería Matemática (iMAT)** en **ICAI**.
Aquí reúno **todas las prácticas, materiales, datasets, cuadernos de trabajo y entregables** que he desarrollado a lo largo del curso, junto con un entorno virtual que permite ejecutar cada proyecto de forma autónoma y reproducible.

Este repositorio refleja tanto mi **progreso técnico** en el análisis y la representación visual de datos como mi **crecimiento personal y metodológico** al aprender a comunicar información compleja de forma clara, rigurosa y estética.

---

## 🎯 Sobre la asignatura

La asignatura de **Visualización de Datos** tiene como objetivo principal enseñar a **comprender y comunicar información a través de los datos**.
En el marco del grado iMAT, combina los fundamentos matemáticos y estadísticos con la programación y el diseño visual, fomentando una mirada crítica hacia cómo percibimos, interpretamos y representamos la información.

Durante el curso se trabajan aspectos como:

* 📈 **Fundamentos del diseño visual**: color, forma, percepción y jerarquía.
* 🧮 **Procesamiento y limpieza de datos** con herramientas como **pandas** y **NumPy**.
* 🧭 **Visualización estática y dinámica** mediante librerías como **Matplotlib**, **Seaborn**, **Plotly** y **Altair**.
* 🔍 **Exploración interactiva de datos** y creación de dashboards.
* 🧠 **Storytelling con datos**: cómo convertir el análisis en una narrativa visual coherente.

Cada práctica busca desarrollar una habilidad concreta —desde dominar la sintaxis de las librerías básicas hasta construir visualizaciones interactivas que comuniquen ideas complejas de forma efectiva y honesta.

## 🧠 Aprendizajes clave

Durante el curso aprendemos que **visualizar datos no consiste solo en graficar**, sino en **pensar visualmente**: entender qué queremos comunicar, elegir las variables adecuadas y construir una representación honesta, legible y significativa.

Algunos aprendizajes esenciales que me llevo son:

* La importancia de la **ética y la transparencia** en la comunicación de datos.
* La **relación entre percepción y comprensión visual**: cómo el diseño guía la interpretación.
* El valor de la **iteración y la experimentación** como parte del proceso analítico.
* La necesidad de **conectar análisis técnico con narrativa visual** para contar historias con impacto.
* Y, sobre todo, que cada error —cada gráfico mal escalado o cada eje mal interpretado— es una oportunidad para aprender a mirar los datos de otra manera.

## ⚙️ Requisitos y uso

Para ejecutar los notebooks o scripts de este repositorio:

1. Clona el repositorio:

   ```bash
   git clone https://github.com/tuusuario/visualizacion-datos-imat.git
   cd visualizacion-datos-imat
   ```
2. Activa el entorno virtual:

   ```bash
   source venv/bin/activate
   ```
3. Instala las dependencias (si no están ya incluidas):

   ```bash
   pip install -r requirements.txt
   ```
4. Abre cualquier práctica en Jupyter Notebook o Visual Studio Code para explorar el código y las visualizaciones.
