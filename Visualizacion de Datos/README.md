# 📊 Visualización de Datos

Bienvenido a la carpeta de **Visualización de Datos** del repositorio iMAT 2, donde reúno todas mis prácticas y el proyecto final de esta asignatura del **Grado en Ingeniería Matemática (iMAT)** en **ICAI**.
Aquí se documentan **prácticas, datasets, notebooks y entregables** desarrollados a lo largo del curso, explorando herramientas de visualización que van desde el análisis estadístico hasta los dashboards interactivos.

---

## 🎯 Sobre la asignatura

La asignatura de **Visualización de Datos** enseña a **comprender y comunicar información a través de los datos**.
Combina fundamentos matemáticos y estadísticos con programación y diseño visual, fomentando una mirada crítica hacia cómo percibimos, interpretamos y representamos la información.

Durante el curso se trabajan aspectos como:

* 📈 **Fundamentos del diseño visual**: color, forma, percepción y jerarquía.
* 🧮 **Procesamiento y limpieza de datos** con herramientas como **pandas** y **NumPy**.
* 🧭 **Visualización estática y dinámica** mediante librerías como **Seaborn**, **Plotly** y herramientas BI como **Tableau**.
* 🔍 **Exploración interactiva de datos** y creación de dashboards.
* 🧠 **Storytelling con datos**: cómo convertir el análisis en una narrativa visual coherente.

## 📚 Contenido de la carpeta

### 📊 [P1 - Tableau](./P1%20-%20Tableau)
*Introducción a la visualización con herramientas BI*
- 🗂️ Ejercicio práctico en **Tableau** (entregable en PDF)
- 📁 Dataset: ventas de videojuegos (`vgsales.xlsx`)
- 🎯 Creación de dashboards e informes visuales sin código

### 🎨 [P2 - Seaborn](./P2%20-%20Seaborn)
*Visualización estadística con Python*
- 📓 Notebooks con análisis exploratorio de datos (EDA)
- 📁 Dataset: empleos en ciencia de datos (`Cleaned_DS_Jobs.csv`)
- 📊 Visualizaciones con **Seaborn** y **Matplotlib**: distribuciones, correlaciones, catplots
- 🌐 Exportación de análisis completo en HTML (`analisis_completo.html`)
- 🛠️ Scripts de configuración del entorno (`setup.sh`, `activar.sh`)

### 🚀 [P3 - Plotly](./P3%20-%20Ploty)
*Visualización interactiva con Python*
- 📓 Notebook principal de **Plotly** (`01_plotly.ipynb`)
- 🧩 Ejercicio guiado para estudiantes (`99_plotly_student_exercise.ipynb`)
- 📈 Gráficos interactivos: scatter plots, mapas, figuras animadas

### 🏆 [Proyecto Final](./Proyecto%20Final)
*Dashboard interactivo como entregable final de la asignatura*
- 🖥️ Aplicación web construida con **Dash** + **Plotly** (`app.py`)
- 🎛️ Dashboard interactivo para exploración y comunicación visual de datos

## ⚙️ Requisitos y uso

Para ejecutar los notebooks o scripts de este repositorio:

1. Clona el repositorio:

   ```bash
   git clone https://github.com/claudialbzz/Visualizacion.git
   cd Visualizacion
   ```

2. Activa el entorno virtual (si está disponible en la práctica):

   ```bash
   source activar.sh
   # o bien
   source env_icai/bin/activate
   ```

3. Instala las dependencias si es necesario:

   ```bash
   pip install seaborn plotly dash pandas jupyter
   ```

4. Abre cualquier práctica en Jupyter Notebook o Visual Studio Code para explorar el código y las visualizaciones.
