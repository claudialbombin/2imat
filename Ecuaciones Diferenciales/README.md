# 📘 Ecuaciones Diferenciales

Bienvenido a mi repositorio, un espacio donde documento mi recorrido por la asignatura de **Ecuaciones Diferenciales** del **Grado en Ingeniería Matemática (iMAT)** en **ICAI**.
Aquí reúno **todas las prácticas, ejercicios, demostraciones, implementaciones numéricas y entregables** que he desarrollado a lo largo del curso.

Este repositorio refleja tanto mi **progreso técnico** en el análisis y resolución de ecuaciones diferenciales como mi **crecimiento personal y metodológico** al aprender a modelizar fenómenos continuos y aplicar herramientas matemáticas a problemas reales.

---

## 🎯 Sobre la Asignatura

La asignatura de **Ecuaciones Diferenciales** tiene como objetivo principal enseñar a **modelizar, analizar y resolver** sistemas que describen la evolución de magnitudes continuas en el tiempo o el espacio.
En el marco del grado iMAT, combina el rigor teórico de las matemáticas con la **aplicación práctica en ingeniería, física y computación científica**, fomentando una comprensión profunda de las herramientas analíticas y numéricas.

Durante el curso se trabajan aspectos como:

* 📈 **Ecuaciones diferenciales ordinarias** de primer y segundo orden
* ⚙️ **Sistemas de ecuaciones diferenciales** y análisis de estabilidad
* 🔢 **Métodos numéricos**: Euler, Runge–Kutta y aproximaciones multietapa
* 🌐 **Ecuaciones en derivadas parciales básicas** y separación de variables
* 🧮 **Transformada de Laplace** y resolución de sistemas lineales

Cada práctica busca desarrollar una habilidad concreta —desde resolver analíticamente una EDO lineal hasta implementar métodos numéricos para aproximar soluciones.

## 🧠 Aprendizajes Clave

Durante el curso he aprendido que **las ecuaciones diferenciales son el lenguaje de la naturaleza**: una forma universal de expresar leyes físicas, procesos biológicos y dinámicas de sistemas complejos.

Algunos aprendizajes esenciales que me llevo son:

* La importancia del **análisis cualitativo** más allá de la solución exacta
* La conexión entre **modelo matemático y fenómeno real**
* El valor de los **métodos numéricos** para problemas sin solución cerrada
* La utilidad de la **transformada de Laplace** en la ingeniería
* Y, sobre todo, que resolver ecuaciones diferenciales es aprender a **pensar en cambio y dinámica**

## ⚙️ Requisitos y Uso

Para ejecutar los códigos y visualizar las simulaciones:

1. Clona el repositorio:

   ```bash
   git clone https://github.com/tuusuario/ecuaciones-diferenciales-imat.git
   cd ecuaciones-diferenciales-imat
   ```

2. Instala las dependencias básicas:

   ```bash
   pip install numpy matplotlib scipy
   ```

3. Ejecuta un ejemplo numérico:

   ```bash
   python Practicas/Principales/P1 - EDOs/solver_rk4.py
   ```

4. Para visualizar los resultados o comparar métodos:

   ```bash
   open Practicas/Principales/P2 - Sistemas/analisis_estabilidad.ipynb
   ```
